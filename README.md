## III. Thiết bị

- ZUBoard 1CG mã XCZU1CG-1SBVA484E
- Module điều khiển Leds and Keys MDU1093 tích hợp ic **TM1638**.
- 3 đường dây tín hiệu STB, CLK, DIO, dây VCC +5V, dây GND.
